Explain how a browser determines what elements match a CSS selector.

Browsers match selectors from rightmost (key selector) to left. Browsers filter out elements in the DOM according to the key selector, and traverse up its parent elements to determine matches. The machine moves through each selector until it finds a match or discards the rule.
